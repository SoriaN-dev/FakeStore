# React TypeScript Next.js Basic

Este repositorio contiene una aplicación básica construida con **React**, **TypeScript** y **Next.js**.

## Estructura del proyecto

- `api/` - Configuración de Axios y llamadas a la API.
- `app/` - Páginas principales y configuración de enrutamiento.
- `components/` - Componentes reutilizables.
- `context/` - Contextos para estado global (carrito y tema).
- `model/` - Modelos de datos (por ejemplo, Producto).
- `public/` - Archivos estáticos.

## Instalación

Primero clona el repositorio y luego instala las dependencias:

```bash
git clone <url-del-repositorio>
pm install # o yarn install o pnpm install
```

## Uso

Para iniciar la aplicación en modo desarrollo:

```bash
pnpm dev
```

Abre `http://localhost:3000` en tu navegador.

## Descripción del proyecto

Aplicación de tienda básica que consume la API de [Fake Store](https://fakestoreapi.com) para mostrar productos, permite filtrarlos y gestionar un carrito de compras.
